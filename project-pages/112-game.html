112 Game

April 2022

This 15112 class term project was created in Python, using CMU Graphics. The project generates a terrain based on the amplitudes of a given audio file, allowing the character to traverse the terrain while the audio plays. A cannon also shoots at the character: if hit, you lose points; if not, you gain. It features accurate physics, Voronoi noise for background generation, and responsive terrain generation.

Programming

Python

About the Project

The 112 Game was developed as a term project for the CMU 15112 class, leveraging Python and the CMU Graphics library to create an interactive gaming experience. The game dynamically generates terrain based on the audio file provided by the user, adding a unique element where the gameplay is influenced by the music's rhythm and amplitude. As the character navigates the terrain, they must avoid projectiles fired from a cannon, adding a layer of challenge and skill to the game.

Key Features

  • Responsive Terrain Generation: Terrain changes dynamically based on audio amplitude, creating a unique gameplay experience for each track.
  • Accurate Physics: The game incorporates realistic physics to ensure smooth character movement and projectile interaction.
  • Voronoi Noise Background: Adds depth and aesthetic appeal to the game's environment, enhancing the visual experience.
  • Scoring System: Points are awarded or deducted based on whether the character avoids or is hit by the cannon projectiles.
